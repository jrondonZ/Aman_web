# Brand assets

The site uses two logo images:

| File              | Logo                  | Used for                                    |
| ----------------- | --------------------- | ------------------------------------------- |
| `aman-logo.png`   | **Horus eye** (Aman)  | nav + footer brand, favicon, social preview |
| `amanix-logo.png` | **Scarab** (Amanix)   | Amanix sections, feature icon, chat avatar  |

Both are framed in a light "seal" medallion in the UI, so they read cleanly on
the dark theme. `aman-logo.png` is the original artwork; `amanix-logo.png` is the
scarab exported to PNG (transparent background) for the web.
